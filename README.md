# qwlabs-skills

A collection of Claude Code skills. Each skill is a self-contained tool that solves a specific problem.

## Installation

```bash
npx -y skills add https://smithery.ai/skills/qwlabs/api-doc
```

No additional configuration is needed after installation.

## Skills

| Skill | Core Capability |
|-------|----------------|
| [api-doc](skills/api-doc/SKILL.md) | Generate single-file HTML API docs from TypeSpec (`.tsp`) definitions |
