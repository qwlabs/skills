## 请求参数
```yaml

  apiName:
    type: string
    description: 请求接口名称
    default: hy_wms_get_transfer_orders_by_document_codes_WebApi
  outType:
    type: string
    description: 请求接口返回格式
    default: json
  data:
    type: array
    description: 单据编码列表
    items:
```
