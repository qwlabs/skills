title: 创建出库单
metadata:
---

## 应用场景
- 业务系统需要进行出库时请求

## 请求前置条件
- 无

## 请求后结果
- 无

## 请求参数
```yaml
type: object
properties:
  apiName:
    type: string
    description: 请求接口名称
    default: hy_wms_create_outbound_order_WebApi
  outType:
    type: string
    description: 请求接口返回格式
    default: json
  data:
    type: object
    description: 出库单数据
    properties:
      confirm:
        type: boolean
        description: 是否自动确认
        default: false
      documentTypeCode:
        type: string
        description: 单据类型编码
      warehouseCode:
        type: string
        description: 出库仓库编码
      businessDocumentTypeCode:
        type: string
        description: 业务单据类型编码
      businessDocumentCode:
        type: string
        description: 业务单据编码
      lines:
        type: array
        description: 出库单行
        items:
          type: object
          properties:
            materialCode:
              type: string
              description: 物料编码
            unit:
              type: string
              description: 数量单位
            lot:
              type: string
              description: 批次号
            sn:
              type: string
              description: 序列号
            qty:
              type: number
              description: 数量
required: [apiName, outType, data]
```

## 请求示例
```json
{
  "apiName": "hy_wms_create_outbound_order_WebApi",
  "outType": "json",
  "data": {
    "confirm": false,
    "documentTypeCode": "OUT001",
    "warehouseCode": "WH01",
    "businessDocumentTypeCode": "SO001",
    "businessDocumentCode": "SO202401010001",
    "lines": [
      {
        "materialCode": "M001",
        "unit": "吨",
        "lot": "LOT20240101",
        "sn": "",
        "qty": 5.0
      }
    ]
  }
}
```
