---
title: 确认出库单
metadata:
---

## 应用场景
- 仓储已经明确可出库后

## 请求前置条件
- 无

## 请求后结果
- 冻结待出库库存

## 请求参数
```yaml
type: object
properties:
  apiName:
    type: string
    description: 请求接口名称
    default: hy_wms_confirm_outbound_order_WebApi
  outType:
    type: string
    description: 请求接口返回格式
    default: json
  data:
    type: object
    description: 确认数据
    properties:
      code:
        type: string
        description: 出库单编码
required: [apiName, outType, data]
```

## 请求示例
```json
{
  "apiName": "hy_wms_confirm_outbound_order_WebApi",
  "outType": "json",
  "data": {
    "code": "OUT202401010001"
  }
}
```

## 响应参数
```yaml
type: object
properties:
  code:
    type: integer
    description: 状态码
  message:
    type: string
    description: 消息
  data:
    type: object
    description: 数据对象
    properties:
      code:
        type: string
        description: 返回状态
      message:
        type: string
        description: 返回提示信息
      data:
        type: object
        description: 确认的出库单信息
        properties:
          code:
            type: string
            description: 出库单编号
          documentTypeCode:
            type: string
            description: 单据类型编码
          warehouseCode:
            type: string
            description: 仓库编码
          businessDocumentTypeCode:
            type: string
            description: 业务单据类型编码
          businessDocumentCode:
            type: string
            description: 业务单据编码
          lines:
            type: array
            description: 出库单行
            items:
              type: object
              properties:
                materialCode:
                  type: string
                  description: 物料编码
                unit:
                  type: string
                  description: 数量单位
                lot:
                  type: string
                  description: 批次号
                sn:
                  type: string
                  description: 序列号
                qty:
                  type: number
                  description: 数量
```
