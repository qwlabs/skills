创建入库单
提供方WMS
接口协议HTTP
请求路径/orbit/modeling/webApi
请求方法POST
请求参数
字段名类型说明必填默认值约束
apiNamestring请求接口名称选填hy_wms_create_inbound_order_WebApi
outTypestring请求接口返回格式选填json
dataobject入库单数据
typestringboolean
descriptionstring是否自动确认
defaultstringfalse
documentTypeCodestring单据类型编码
warehouseCodestring入库仓库编码
businessDocumentTypeCodestring业务单据类型编码
businessDocumentCodestring业务单据编码
linesnumber数量
请求示例
{
  "apiName": "hy_wms_create_inbound_order_WebApi",
  "outType": "json",
  "data": {
    "confirm": false,
    "documentTypeCode": "IN001",
    "warehouseCode": "WH01",
    "businessDocumentTypeCode": "PO001",
    "businessDocumentCode": "PO202401010001",
    "lines": [
      {
        "materialCode": "M001",
        "unit": "吨",
        "lot": "LOT20240101",
        "sn": "",
        "qty": 10.5
      }
    ]
  }
}
响应参数
字段名类型说明必填默认值约束
codeinteger状态码
messagestring消息
dataobject数据对象
typestringstring
descriptionstring返回状态 (200:成功, 400:失败)
messagestring返回提示信息
datanumber数量
响应示例
{
  "code": 200,
  "message": "success",
  "data": {
    "code": "200",
    "message": "创建成功",
    "data": {
      "code": "IN202401010001",
      "documentTypeCode": "IN001",
      "warehouseCode": "WH01",
      "businessDocumentTypeCode": "PO001",
      "businessDocumentCode": "PO202401010001",
      "lines": [
        {
          "materialCode": "M001",
          "unit": "吨",
          "lot": "LOT20240101",
          "sn": "",
          "qty": 10.5
        }
      ]
    }
  }
}
